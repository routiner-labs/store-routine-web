# 알바생 업무·출퇴근·매장 루틴 관리 서비스 기획서

## 1. 서비스 개요

본 서비스는 소상공인 사장과 알바생 사이에서 발생하는 반복적인 업무 지시, 출퇴근 확인, 오픈·마감 체크, 특별 요청, 자료 전달을 하나의 앱에서 관리하는 매장 운영 관리 서비스입니다.

단순한 출퇴근 관리 앱이 아니라, 알바생이 출근했을 때 오늘 해야 할 업무와 수행 방법을 확인하고, 업무 완료 여부를 체크하거나 사진·숫자·메모로 증빙하며, 사장은 미완료 업무와 이상 상황만 확인할 수 있도록 돕는 것을 목표로 합니다.

---

## 2. 핵심 문제

소규모 매장에서는 사장과 알바생 간 업무가 대부분 카카오톡, 구두 설명, 종이 메모, 개인 기억에 의존합니다.

이로 인해 다음과 같은 문제가 발생합니다.

- 알바생이 오늘 해야 할 일을 정확히 모릅니다.
- 신입 알바생에게 같은 업무 설명을 반복해야 합니다.
- 오픈·마감 업무가 누락될 수 있습니다.
- 불 끄기, 문 잠금, 가스 밸브 확인 같은 중요 업무가 제대로 수행됐는지 확인하기 어렵습니다.
- 특별 지시가 카카오톡 대화에 묻힙니다.
- 알바생의 요청사항이 정리되지 않습니다.
- 사장이 매장에 없는 시간대의 업무 상태를 확인하기 어렵습니다.
- 출퇴근, 업무 수행, 요청, 자료 확인이 각각 다른 채널에 흩어집니다.

---

## 3. 서비스의 핵심 가치

이 서비스의 핵심 가치는 다음과 같습니다.

> 알바생이 “무엇을 해야 하는지”와 “어떻게 해야 하는지”를 앱에서 확인하고,  
> 사장은 “했는지”와 “문제 있는지”만 확인할 수 있게 합니다.

즉, 사장과 알바생 사이의 반복 연락을 줄이고, 매장 운영 업무를 표준화하는 것이 핵심입니다.

## 4. 제품 포지션

### 4.1 잘못된 포지션

- 알바 출퇴근 관리 앱
- 급여 계산 앱
- 일반 체크리스트 앱
- 직원 스케줄 앱

위 포지션은 기존 서비스와 경쟁이 강하고 차별성이 약합니다.

### 4.2 적합한 포지션

- 사장님이 없어도 매장이 돌아가게 하는 매장 루틴 관리 앱
- 알바생에게 매번 설명하지 않아도 되는 업무 매뉴얼 앱
- 오픈·마감 업무 누락을 막는 알바 업무 체크 앱
- 출퇴근, 체크리스트, 특별 지시, 요청을 한 번에 관리하는 매장 운영 앱

- 재고 확인

```text
오늘 근무
16:00 ~ 22:00

오늘 할 일
[ ] 오픈 체크리스트
[ ] 냉장고 온도 확인
[ ] 재고 부족 품목 확인
[ ] 특별 지시 1건
```

- 오픈 체크리스트

- 위생 점검 체크리스트
- 일회성 체크리스트

### 체크리스트 항목 예시

- 매장 조명 켜기
- 포스 전원 확인
- 준비금 확인
- 냉장고 온도 확인

- 포스 마감
- 특이사항 작성

## 7.4 수행 방법 안내

각 체크리스트 항목에는 수행 방법을 작성할 수 있습니다.

단순히 “문 잠금 확인”이라고 쓰는 것이 아니라, 알바생이 실제로 어떻게 해야 하는지를 안내합니다.

### 예시

```text
업무명: 출입문 잠금 확인

수행 방법:
1. 출입문을 잠근다.
2. 손잡이를 당겨 문이 열리지 않는지 확인한다.
3. 잠금 상태가 보이도록 사진을 찍는다.

완료 방식:
사진 첨부 필수
```

이 기능은 신입 알바생 교육과 업무 표준화에 중요합니다.

---

## 7.5 완료 방식 설정

체크리스트 항목마다 완료 방식을 다르게 설정할 수 있습니다.

| 완료 방식 | 설명 | 예시 |
|---|---|---|
| 체크 | 단순 완료 처리 | 조명 끄기 |
| 사진 첨부 | 사진을 찍어야 완료 | 문 잠금, 가스 밸브 |
| 숫자 입력 | 숫자 값을 입력해야 완료 | 냉장고 온도, 준비금 |
| 메모 입력 | 내용을 작성해야 완료 | 특이사항, 고객 클레임 |
| 선택값 | 정상/이상 있음 선택 | 설비 점검 |
| 사장 확인 | 사장이 확인해야 최종 완료 | 현금 정산 |

## 7.6 마감 체크리스트

마감 체크리스트는 서비스의 핵심 기능 중 하나입니다.

### 마감 체크리스트 예시

```text
1. 포스 마감
- 수행 방법: 포스에서 일마감 메뉴를 실행하고 매출과 현금을 확인한다.
- 완료 방식: 금액 입력 + 사진 첨부

2. 현금함 정리
- 수행 방법: 준비금을 제외한 나머지 금액을 정리한다.
- 완료 방식: 숫자 입력

3. 냉장고 문 확인
- 수행 방법: 냉장고 문이 완전히 닫혔는지 확인한다.
- 완료 방식: 체크

4. 가스 밸브 확인
- 수행 방법: 가스 밸브가 잠금 방향으로 되어 있는지 확인한다.
- 완료 방식: 사진 첨부

5. 매장 조명 끄기
- 수행 방법: 홀, 주방, 화장실, 창고 조명을 모두 끈다.
- 완료 방식: 체크

6. 출입문 잠금
- 수행 방법: 출입문을 잠근 뒤 손잡이를 당겨 확인한다.
- 완료 방식: 사진 첨부

7. 특이사항 작성
- 수행 방법: 파손, 재고 부족, 고객 클레임, 설비 이상이 있으면 작성한다.
- 완료 방식: 메모 입력
```

퇴근 전 필수 마감 항목이 남아 있으면 알바생에게 안내합니다.

```text
아직 완료하지 않은 마감 업무가 있습니다.

- 출입문 잠금 확인
- 가스 밸브 사진 첨부
- 특이사항 작성

[마감 업무 완료하기]
[사유 작성 후 퇴근]
```

완전히 퇴근을 막기보다는, 사유 작성 후 퇴근할 수 있도록 하는 방식이 적절합니다.

- 냉장고 2번 칸 정리

- 재고 부족 품목 사진 찍기
- 특정 고객 요청사항 확인

### 특별 지시 속성

- 제목
- 상세 내용
- 대상 알바생
- 기한
- 완료 방식
- 사진 필요 여부
- 사장 확인 필요 여부

## 7.8 요청함

알바생은 앱을 통해 사장에게 요청사항을 남길 수 있습니다.

카카오톡으로 흩어지는 요청을 앱에서 관리하는 것이 목적입니다.

### 요청 유형

| 요청 유형 | 예시 |
|---|---|
| 재료/비품 부족 | 우유, 컵, 빨대, 세제 부족 |
| 장비 고장 | 포스, 냉장고, 커피머신, 조명 문제 |
| 근무 변경 | 지각, 조퇴, 대타, 휴무 요청 |
| 고객 이슈 | 클레임, 분실물, 환불 요청 |
| 청소/시설 | 화장실, 매장, 창고 문제 |
| 기타 요청 | 분류되지 않는 요청 |

### 요청 상태

- 요청됨
- 확인됨
- 처리중
- 완료
- 반려

### 예시

```text
요청 유형: 재료 부족
내용: 우유가 2팩 남았습니다.
사진: 냉장고 사진 첨부
상태: 확인됨
```

## 7.10 캘린더

사장은 캘린더에서 알바생 출근 시간을 한눈에 볼 수 있습니다.

단순 근무표가 아니라, 사장이 매장에 있어야 할 시간을 판단하는 데 사용합니다.

### 주요 기능

- 알바생 근무 일정 조회
- 근무 공백 시간 표시
- 사장 부재시간 표시
- 혼자 근무하는 시간 표시
- 알바생별 근무 시간 확인

### 예시

```text
화요일
10:00~14:00 김민수
18:00~22:00 이지은

공백 시간:
14:00~18:00
```

이 경우 사장은 14:00~18:00에 직접 매장에 있어야 하는지 판단할 수 있습니다.

## 8. 사장용 화면

사장은 모든 내용을 자세히 보는 것이 아니라, 미완료와 이상 상황을 빠르게 확인해야 합니다.

### 사장 홈 예시

```text
오늘 매장 상태

출근
- 김민수 출근 완료
- 이지은 16:00 출근 예정

업무
- 오픈 체크리스트 8/10 완료
- 특별 지시 1/2 완료
- 마감 체크리스트 대기

요청
- 재료 부족 1건
- 대타 요청 1건
- 장비 이상 0건

확인 필요
- 냉장고 온도 9도
- 출입문 잠금 사진 미첨부
```

사장 홈의 핵심은 전체 업무 목록이 아니라, 문제 있는 항목만 보여주는 것입니다.

## 9. 알바생용 화면

알바생 화면은 단순해야 합니다.

### 알바생 홈 예시

```text
오늘 근무
16:00 ~ 22:00

오늘 할 일
[ ] 오픈 체크리스트
[ ] 재고 확인
[ ] 특별 지시 1건

요청하기
[재료 부족]
[장비 고장]
[대타 요청]

자료 보기
[포스 사용법]
[마감 방법]
```

- 출근
- 오늘 할 일 확인
- 완료 체크
- 요청 등록
- 퇴근

## 10. MVP 범위

1차 MVP는 기능을 과하게 넣지 않고 핵심 가치 검증에 집중합니다.

### 10.1 1차 MVP 필수 기능

- 사장 계정 생성
- 매장 생성
- 알바생 초대
- 알바생 출근/퇴근
- 체크리스트 생성
- 체크리스트 항목별 수행 방법 작성
- 완료 방식 설정
  - 체크
  - 사진 첨부
  - 숫자 입력
  - 메모 입력
- 오픈/마감 체크리스트
- 특별 지시

## 13. 차별화 포인트

### 13.1 수행 방법이 있는 체크리스트

단순 체크리스트가 아니라 업무 수행 방법이 포함됩니다.

```text
업무명: 냉장고 온도 확인
수행 방법: 냉장고 상단 온도계를 확인하고 5도 이하인지 입력하세요.
완료 방식: 숫자 입력
```

### 13.2 사진/숫자/메모 기반 증빙

체크만 하는 것이 아니라 업무 성격에 맞게 증빙을 남깁니다.

### 13.3 사장 부재시간 관리

사장이 매장에 없는 시간에도 알바생이 업무를 수행했는지 확인할 수 있습니다.

### 13.4 신입 알바 교육 대체

업무 수행 방법이 쌓이면 신입 알바생 교육 자료로 사용할 수 있습니다.

### 13.5 카카오톡 업무 연락 감소

특별 지시, 요청, 업무 자료를 앱에서 처리하여 카카오톡에 업무 내용이 묻히는 문제를 줄입니다.

## 18. 운영 리스크

### 18.1 기존 서비스와의 경쟁

출퇴근·급여 관리 서비스는 이미 존재합니다.  
따라서 출퇴근 앱으로 포지셔닝하면 차별화가 어렵습니다.

대응 방향:

```text
근태관리보다 매장 루틴 관리와 업무 누락 방지에 집중한다.
```

### 18.2 소상공인 가격 민감도

소상공인은 월 사용료에 민감합니다.

대응 방향:

```text
월 4,900원~9,900원 수준의 낮은 진입 가격을 제공한다.
```

### 18.3 알바생 사용 저항

알바생이 귀찮다고 느끼면 도입이 어렵습니다.

대응 방향:

```text
알바생 화면은 출근, 오늘 할 일, 체크, 요청, 퇴근만 제공한다.
```

### 18.4 급여 계산 리스크

급여 계산은 법정수당, 휴게시간, 주휴수당 등 법적 이슈가 있습니다.

대응 방향:

```text
초기에는 급여 자동 계산을 제외하고 근무시간 기록까지만 제공한다.
```

### 18.5 사장 초기 설정 부담

사장이 체크리스트를 처음부터 만들기 어려울 수 있습니다.

대응 방향:

```text
업종별 템플릿을 제공한다.
```

```text
1. 사장이 매일 앱을 확인하는가

5. 요청함을 통해 사장과 알바생의 반복 연락이 줄어드는가
6. 최소 5개 매장에서 2주 이상 지속 사용되는가
7. 월 4,900원~9,900원 결제 의사가 있는가
```

## 21. 최종 서비스 정의

본 서비스는 알바생 출퇴근, 오늘 업무, 업무 수행 방법, 오픈·마감 체크리스트, 특별 지시, 요청사항을 하나로 묶어 관리하는 소상공인용 매장 루틴 관리 앱입니다.

핵심 문장은 다음과 같습니다.

```text
알바생은 앱에서 오늘 할 일과 수행 방법을 보고,
사장은 미완료와 이상 상황만 확인합니다.
```

이 서비스는 단순한 근태관리 앱이 아니라, 사장과 알바생 사이의 반복 커뮤니케이션을 줄이고 매장 운영 업무를 표준화하는 것을 목표로 합니다.
