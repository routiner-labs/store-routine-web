```
src/
```

- `src/app/employee/EmployeeNav.tsx` — 동일

- `src/app/owner/page.tsx` — alertTitle 이모지 제거

- `.claude/CLAUDE.md` — 예시 코드 ❌/✅ 기호 -> [Bad]/[Good] 텍스트로 교체

주요 결정:
- 네비게이션 아이콘 없이 텍스트만 사용. 태블릿 사이드바(72px)에서는 단축 라벨(2자), 데스크톱/모바일에서는 전체 라벨 표시.
- 체크리스트 완료 타입은 이모지 대신 한국어 단어(확인, 사진, 숫자, 메모, 선택, 사장확인)로 표시.
- ✓ 체크마크(U+2713)는 이모지가 아닌 유니코드 기호이므로 체크버튼에서 유지.
